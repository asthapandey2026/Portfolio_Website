#step-by-step necessary commands to run on terminal in order to set up and run this project on any system.

#Install Node.js: Ensure Node.js is installed. Check by running:
node -v
npm -v

#Install Dependencies: Navigate to your project directory and run the following command on terminal:
npm install

#Install React Icons(if not already installed): Since we are using react-icons for icons, we need to ensure that the package is installed. If it's not installed, run the following command:
npm install react-icons

#Ensure React is Installed: Ensure that react and react-dom are installed if they aren't already:
npm install react react-dom

#Start the Development Server:
npm start

#The app will start on http://localhost:3000 by default.
#This setup will ensure this project runs on any system.
